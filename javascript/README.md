# JavaScript Examples for GitHub Copilot Training

This directory contains JavaScript/Node.js examples that demonstrate various programming concepts and how GitHub Copilot can assist with JavaScript development.

## 📁 Files

- `basic_functions.js` - Basic function examples with clear naming
- `async_patterns.js` - Async/await and Promise examples
- `dom_manipulation.js` - DOM manipulation examples
- `data_processing.js` - Array and object manipulation
- `api_examples.js` - API integration examples
- `modern_es6.js` - Modern JavaScript features

## 🚀 Getting Started

1. Open any JavaScript file in this directory
2. Try running the examples with Node.js: `node filename.js`
3. Experiment with different prompting techniques
4. Add your own functions using descriptive names

## 💡 Tips for JavaScript with Copilot

- Use JSDoc comments: `/** @param {string} name */`
- Write descriptive function and variable names
- Use modern ES6+ syntax
- Be explicit about async/await patterns
- Include example usage in comments

## 🛠️ Setup for Node.js Examples

```bash
npm init -y
npm install axios # for API examples
```
