# dkm_github_copilot_training
Exercise: Code with Codespaces

## Getting Started with GitHub Codespaces

This repository is configured for GitHub Codespaces to provide a ready-to-use development environment for GitHub Copilot training.

### Opening in Codespaces

1. Click the green **Code** button on the repository page
2. Select the **Codespaces** tab
3. Click **Create codespace on main** (or your current branch)

### What's Included

The Codespaces environment includes:

- **GitHub Copilot** and **GitHub Copilot Chat** extensions
- **Python 3.11** runtime
- **Node.js LTS** runtime
- **Git** and **GitHub CLI** tools
- **Jupyter** notebook support
- Common VS Code extensions for development

### Ready to Code!

Once your Codespace loads, you'll have a fully configured development environment with GitHub Copilot ready to assist you with coding exercises and learning.
